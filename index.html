Chapter 03
🎯 Prompt Engineering — AI Se Sahi Kaam Karwao
Ye sabse important skill hai — 80% result prompt pe depend karta hai

Prompt wo instruction hai jo aap AI ko dete ho. Jitna achha prompt = utna achha result. Ye ek learnable skill hai — practice se perfect hoti hai.

Bad Prompt vs Good Prompt
❌ Bad (vague): "Mujhe ek website banao"
AI confused — kaunsi website? Kis liye? Features kya? Design kaisa?
✅ Good (specific): "Mujhe ek educational website banao coaching institute ke liye — Sanchit Classes, Dhanbad. Features: MCQ test (20 random questions), Result page with score analysis, Flashcards. Design: Navy blue + orange theme, mobile friendly. Pure HTML CSS JS — offline kaam kare."
Perfect Prompt Formula — 5 Elements
1
🎭 Role batao
"Tu ek expert web developer hai jo educational websites banata hai..." — Role dene se AI us perspective se sochta hai.
2
📋 Clearly describe karo kya chahiye
Features list karo — MCQ test, 75+ questions, 4 difficulty levels, topic-wise analysis, etc.
3
📁 Format specify karo
"Pure HTML file mein do", "CSV format mein do", "JSON mein do" — output format clear karo.
4
🚫 Restrictions batao
"Mobile friendly hona chahiye", "Koi external CDN link nahi", "Offline kaam kare", "File 1MB se kam".
5
💡 Example do (optional but powerful)
"Kuch aisa dikhna chahiye jaise Khan Academy ya BYJU's" — reference dene se AI samajhta hai.
Ready-Made Prompts Library — Copy Karke Use Karo
🌐 Website Banana:
"Mujhe ek complete [SUBJECT] learning website banao Class [X] CBSE ke liye.
Features: (1) Notes section — topic-wise cards mein (2) MCQ test — 50+ questions, 4 categories Easy/Medium/Hard/Board (3) Flashcards — flip animation (4) Result analysis — topic-wise chart.
Design: Navy blue + Orange theme, mobile friendly, professional.
Tech: Pure HTML CSS JS — offline kaam kare, koi external link nahi, single file.
Language: Hinglish (Hindi + English mix)."
📊 Questions Generate Karna (Bulk):
"Generate 50 MCQ questions on [TOPIC] for [SSC CGL/CTET/Banking/UPSC].
Format: CSV mein do.
Columns: id, difficulty(easy/medium/hard/board), topic, question, opt1, opt2, opt3, opt4, ans(0-3), explanation.
Rules: (1) Ek clearly sahi answer (2) Wrong options believable but clearly wrong (3) Explanation mein rule mention karo (4) Agar unsure ho to ⚠️ lagao."
🐛 Error Fix Karna:
"Neeche diya hua code kaam nahi kar raha.
Problem: [EXACT ERROR MESSAGE YA PROBLEM DESCRIBE KARO]
Code dekho → galti dhundho → fix karo → explain karo kya galat tha.
[APNA POORA CODE PASTE KARO]"
🎨 Design Improve Karna:
"Is website ka design improve karo. Ye cheezein fix karo:
(1) Mobile pe text bahut chota dikhraha hai — fix karo
(2) Buttons ka color change karo — orange use karo
(3) Header aur footer add karo — professional lagae
(4) Loading animation add karo
Pure HTML CSS JS mein raho."
📝 Content Likhwana:
"[TOPIC] pe ek detailed article likho [TARGET AUDIENCE] ke liye.
Language: Hindi/English/Hinglish
Length: ~500 words
Include: Introduction, 3 main points with examples, Conclusion
Tone: Simple, friendly, educational"
🔄 Iterate/Improve Karna:
"Is output mein ye changes karo:
(1) [Specific change 1]
(2) [Specific change 2]
(3) [Specific change 3]
Baki sab same rakhna — sirf ye teen cheezein change karo."
🔑 Pro Tip — Chain Prompting: Ek baar mein sab mat maango. Pehle basic banwao → phir feature add karo → phir design improve karo. "Chain prompting" mein har step pe better output milta hai!

Chapter 04
🌐 Pehli Website Banao — 30 Minutes Mein
Aaj hi banao — koi background needed nahi
❓ Common Doubt: Kya Code Samajhna Zaroori Hai?

Short answer: Nahi — lekin thoda samajhna helpful hai.

✅ Copy karo directly: Jab website banana ho, result dikhao, quickly deploy karna ho
✅ Samjho thoda: Jab error aaye, kuch change karna ho, client question poochhe
✅ Claude se samjhwao: "Ye code mujhe simple Hindi mein explain karo" — ye best approach hai

Real talk: Sanchit Classes ka poora platform bina ek line code likhe banaya gaya — sirf AI prompts se. Ye guide bhi 2780+ lines AI ne likhi!
1
Claude.ai pe account banao
Browser → claude.ai → Sign Up → Google se login karo → Free plan → Done!
2
Prompt type karo
"Mujhe ek simple coaching institute website banao — Sanchit Classes, Dhanbad. Features: Home, About, Subjects (Math/Science/English/Economics for Class 6-12), Contact with WhatsApp button. Design: Professional, navy blue orange theme, mobile friendly. Pure HTML CSS JS."
3
Code copy karo
Claude code dega → Code block ke upar "Copy" button click karo → Ctrl+C
4
Notepad mein save karo
Notepad open karo (Windows search: "notepad") → Paste karo (Ctrl+V) → File → Save As → Naam: "index.html" → Save type: "All files" → Desktop pe save karo
5
Browser mein open karo
Desktop pe "index.html" dhundho → Double click → Chrome/Firefox mein website khul jayegi! 🎉
6
Improve karo
Kuch acha nahi laga? Claude ko bolo: "Address change karo: 123 Main Road, Dhanbad", "Phone number add karo: 9876543210", "Color thoda lighter karo" — iterate karo!
⚠️ Most Common Mistake — Dhyaan Se Padho!
Notepad mein Save As karte waqt:
✅ File name: index.html (html extension zaroori hai)
✅ Save as type: "All Files (*.*)" — ye select karna MUST hai
❌ Agar "Text Documents (*.txt)" selected hai → file "index.html.txt" save hogi → browser mein open nahi hogi!
VS Code use karo — ye problem automatically solve ho jaati hai!

HTML — Website Ka Dhaancha
Tag
<div>, <p>, <h1>
HTML ka basic building block. Opening <tag> aur closing </tag> hota hai. Jaise brick = ek tag.
div
<div> = Division
Ek container box — elements group karne ke liye. Akela kuch nahi karta — CSS se style dete hain.
class / id
class="btn" id="header"
Element ko naam dena — CSS styling ya JS se pakadne ke liye. class = group naam, id = unique naam.
style attribute
style="color:red"
Directly element pe design lagao. Quick fix ke liye — lekin CSS file better hoti hai.
src / href
src="image.jpg"
File ka path ya link. src = image/script ka source. href = link ka destination.
onclick
onclick="function()"
Button click hone pe kya ho — ye instruction. "Jab click karo tab ye kaam karo."
CSS — Website Ka Design
selector
.class, #id, div
Kis element pe style lagana hai — ye batata hai. .card = card class ke sab elements.
property: value
color: red; font-size: 16px
Property = kya change karna (color, size, margin). Value = kitna/kaunsa (red, 16px, bold).
margin / padding
margin: 10px; padding: 5px
margin = element ke bahar space. padding = element ke andar space. Dono alag hain!
flexbox / grid
display: flex; display: grid
Elements ko row/column mein arrange karne ke tools. Mobile-friendly layouts ke liye essential.
media query
@media(max-width:600px)
Mobile pe alag design — "agar screen 600px se chhhoti ho to ye style lagao." Mobile friendly ka secret!
:root variables
--primary: #E8590C
Color/font ek jagah define karo — poori website mein use karo. Change karna ho to ek jagah change karo.

function
function doThis() { }
Ek kaam karne ki recipe. Ek baar likhو — baar baar call karo. Jaise recipe = function.
variable
let name = "Sanchit"
Data store karne ki jagah. naam = box, value = box ke andar rakhi cheez.
array
let fruits = ["apple","mango"]
Ek saath multiple cheezein store karna — list ki tarah. Questions ka list = array.
JSON
{"name":"Ram","age":25}
Data ka ek format — key:value pairs mein. Database se website tak data bhejne ke liye.
API
Application Programming Interface
Do software ke beech communication — jaise waiter (order leta hai, kitchen tak pohonchata hai, khana wapas laata hai).
console.log
console.log("test")
Debug karne ka tool — browser mein F12 → Console tab mein message dikhata hai. Error dhundne ke liye use karo.
